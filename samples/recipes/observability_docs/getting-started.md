# Getting Started with Nimbus

Nimbus is an observability platform for collecting **metrics**, **logs**, and
**traces**. This guide gets you from zero to your first dashboard.

1. Install the agent (see [Installation](./installation.md)).
2. Send data using the [Ingest API](./ingest-api.md).
3. Build a [Dashboard](./dashboards.md).

For production hardening, read the [Operations Guide](./operations.md).
